  <name>Task 1: Keyboard input handler and letter matcher</name>
  <files>src/systems/input.js, src/systems/matcher.js</files>
  <action>
  **Part A — src/systems/input.js:**

  Create with two exports:
  1. `initInput()` — adds `keydown` and `keyup` window event listeners
  2. `destroyInput()` — removes both listeners (for cleanup/testing)

  Keydown handler logic:
  - If `e.repeat` is true, return immediately (prevent held-key spam)
  - Normalize: `const key = e.key.toUpperCase()`
  - If key is a single character A-Z (`key.length === 1 && key >= 'A' && key <= 'Z'`):
    - Add to `pressed` Set (track held keys)
    - Emit: `events.emit('KEY_PRESS', { key })`

  Keyup handler logic:
  - Normalize key, delete from `pressed` Set

  Use a module-level `const pressed = new Set()` for repeat prevention.
  Do NOT call `preventDefault()` — avoid breaking browser shortcuts (Ctrl+R, etc.).
  Import events from `../core/events.js`.

  **Part B — src/systems/matcher.js:**

  Create with one export:
  1. `initMatcher(obstaclePool)` — subscribes to `KEY_PRESS` event and performs danger zone matching

  Listener logic on `KEY_PRESS`:
  - Compute danger zone: `dangerLeft = getWidth() * GAME.DANGER_ZONE_START`, `dangerRight = getWidth()` (right edge)
  - Get active obstacles from pool: `obstaclePool.getActive()`
  - Iterate obstacles in REVERSE order (closest to player = rightmost first for tie-breaking):
    - If `obs.letter === key` AND `obs.x >= dangerLeft` AND `obs.x <= dangerRight`:
      - Emit `events.emit('OBSTACLE_DESTROYED', { letter: obs.letter, x: obs.x, y: obs.y })`
      - Release obstacle: `obstaclePool.release(obs)`
      - Return immediately (only one obstacle destroyed per keypress)
  - If no match found: do nothing (no penalty in Phase 2)

  Import GAME from config.js, getWidth from core/canvas.js, events from core/events.js.
  </action>

    updateGround(ground, dt, GAME.SCROLL_SPEED);

  <action>
1. Add AUDIO section to src/config.js:
```js
export const AUDIO = {
  MASTER_VOLUME: 0.4,
  SFX_VOLUME: 0.5,
  MUSIC_VOLUME: 0.15,
  POP: { freq: 880, duration: 0.08, type: 'sine' },
  THUD: { freq: 120, duration: 0.2, type: 'sine' },
  GAME_OVER: { startFreq: 440, endFreq: 110, duration: 0.6, type: 'sawtooth' },
  LEVEL_UP: { notes: [523, 659, 784, 1047], noteDuration: 0.1, type: 'square' },
  MUSIC: { baseFreq: 110, filterFreq: 800 }
};
```

2. Create src/systems/audio.js with:
- Module-level `audioCtx` (null until first interaction)
- Module-level `masterGain`, `sfxGain`, `musicGain` nodes
- `createAudioSystem()`:
  - Listen for first 'keydown' on window to create AudioContext and resume it (AUD-05)
  - Once resumed: create gain node chain (master -> sfx, master -> music)
  - Set musicGain.gain.value = 0 (muted by default, AUD-04)
  - Set a module flag `audioReady = true`
  - Also listen for STATE_CHANGE events: on GAME_OVER stop music, on PLAYING start if toggled on

- `playPop()` — AUD-01:
  - Guard: if (!audioReady) return
  - Create oscillator (sine, 880Hz), connect to sfxGain
  - GainNode with exponentialRampToValueAtTime for quick decay (0.08s)
  - Start immediately, stop after duration

- `playThud()` — AUD-02:
  - Guard: if (!audioReady) return
  - Create oscillator (sine, 120Hz), connect to sfxGain
  - GainNode with exponentialRampToValueAtTime for medium decay (0.2s)
  - Start immediately, stop after duration

- `playGameOver()` — AUD-03:
  - Guard: if (!audioReady) return
  - Create oscillator (sawtooth), connect to sfxGain
  - frequency.linearRampToValueAtTime from 440 to 110 over 0.6s
  - GainNode with exponentialRampToValueAtTime for decay
  - Start immediately, stop after 0.6s

- `playLevelUp()` — AUD-06:
  - Guard: if (!audioReady) return
  - Create 4 oscillators (square) playing ascending notes [C5, E5, G5, C6]
  - Each starts 0.1s after the previous (arpeggio)
  - Each has short 0.08s decay
  - Connect all to sfxGain

- Background music (AUD-04):
  - `startMusic()` (internal): create low-frequency oscillator (sawtooth, 110Hz) through BiquadFilter (lowpass, 800Hz), connect to musicGain. Store reference for stop.
  - `stopMusic()` (internal): disconnect and stop music oscillator
  - `toggleMusic()` (exported): flip musicEnabled flag. If enabling: set musicGain.gain to MUSIC_VOLUME and call startMusic(). If disabling: set musicGain.gain to 0 and call stopMusic().
  - `isMusicPlaying()` (exported): return musicEnabled state

- Important: All oscillators use `oscillator.stop(audioCtx.currentTime + duration)` to auto-cleanup. Never accumulate nodes.

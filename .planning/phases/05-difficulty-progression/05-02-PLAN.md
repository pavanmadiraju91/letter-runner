
```js
export function updateSpawner(spawner, dt, difficultyParams, groundY)
```

Where `difficultyParams` is the object returned by `getDifficultyParams()`:
`{ scrollSpeed, spawnInterval, maxObstacles, multiplier, tallObstacles }`

Inside updateSpawner:
1. Replace `GAME.SPAWN_INTERVAL` check with `difficultyParams.spawnInterval`
2. Replace `GAME.MAX_OBSTACLES` check with `difficultyParams.maxObstacles`
3. When configuring acquired obstacle:
   - `obs.speed = difficultyParams.scrollSpeed`
   - If `difficultyParams.tallObstacles`: randomly make ~40% of obstacles taller:
     ```js
     const isTall = difficultyParams.tallObstacles && Math.random() < 0.4;
     obs.height = isTall ? GAME.OBSTACLE_HEIGHT * 1.5 : GAME.OBSTACLE_HEIGHT;
     obs.y = groundY - obs.height;
     ```
   - Otherwise: `obs.height = GAME.OBSTACLE_HEIGHT; obs.y = groundY - obs.height;`

Keep the MIN_OBSTACLE_GAP guard using `GAME.MIN_OBSTACLE_GAP` (this is a display concern, not difficulty).
Keep the unique letter selection logic unchanged (already works correctly).

Remove the import of `GAME.SPAWN_INTERVAL` and `GAME.MAX_OBSTACLES` usage — still import GAME for OBSTACLE_HEIGHT, OBSTACLE_WIDTH, MIN_OBSTACLE_GAP.

<task type="auto">
  <name>Task 2: Wire difficulty system into main.js game loop</name>
  <files>src/main.js</files>
  <action>
Modify main.js to integrate the difficulty system:

1. Add imports at the top:
   ```js
   import { createDifficulty, resetDifficulty, getDifficultyParams } from './systems/difficulty.js';
   ```

2. After `createScore()` / `resetScore()`, add:
   ```js
   createDifficulty();
   resetDifficulty();
   ```

3. In the `update(dt)` function, replace hardcoded GAME.SCROLL_SPEED with dynamic values:
   ```js
   function update(dt) {
     if (getState() !== STATES.PLAYING) return;

     const params = getDifficultyParams();
     updateGround(ground, dt, params.scrollSpeed);
     const groundY = getHeight() - GAME.GROUND_HEIGHT;
     updateSpawner(spawner, dt, params, groundY);
     updateObstacles(obstaclePool, dt);
     cleanupOffscreen(obstaclePool);
     updateScore(dt);
   }
   ```

4. In the `restartGame()` function, add `resetDifficulty()` call:
   ```js
   function restartGame() {
     resetLives();
     resetScore();
     resetDifficulty();
     obstaclePool.getActive().slice().forEach(o => obstaclePool.release(o));
     spawner.timer = 0;
     requestRestart();
   }
   ```

This means:
- Ground scrolls at dynamic speed (visual speed matches obstacle speed)
- Spawner uses current level's spawnInterval, maxObstacles, scrollSpeed, tallObstacles
- Restarting resets difficulty to level 1
- The LEVEL_UP event (emitted by difficulty.js) is already listened to by score.js, input.js, and hud.js from previous phases
  </action>

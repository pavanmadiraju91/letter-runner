- 2px wide legs (mid-stride pose)
- 4-5px wide torso
- 3-4px head with single white eye pixel
- Arms in running pose

Keep `createPlayer` and `resetPlayer` unchanged. Only modify `renderPlayer`.

Important: use `Math.floor` for pixel coordinates to keep edges crisp (imageSmoothingEnabled is already false per D-0102-3).

<task type="auto">
  <name>Task 3: Style obstacles as neon-bordered blocks with glow</name>
  <files>src/entities/obstacle.js</files>
  <action>
Replace the current plain `fillRect` + white text rendering in `renderObstacles` with styled blocks:

1. Import `COLORS` from config.js
2. For each obstacle, draw in this order:
   a. Set `ctx.shadowColor = COLORS.PALETTE.OBSTACLE_GLOW` and `ctx.shadowBlur = 8` for neon glow
   b. Fill the body: `ctx.fillStyle = COLORS.PALETTE.OBSTACLE_BODY` and fillRect
   c. Draw a 2px border: `ctx.strokeStyle = COLORS.PALETTE.OBSTACLE_BORDER`, `ctx.lineWidth = 2`, `ctx.strokeRect(x+1, y+1, w-2, h-2)`
   d. Reset shadow: `ctx.shadowBlur = 0`
3. For the letter:
   a. `ctx.fillStyle = COLORS.PALETTE.OBSTACLE_LETTER`
   b. Use `'bold 22px monospace'` font (slightly smaller for the styled block)
   c. Center text in the block as before

The result: dark blocks with a glowing magenta border and a white letter inside. The glow gives a neon feel.

Keep all other functions (createObstacleFactory, updateObstacles, cleanupOffscreen) unchanged.
  </action>

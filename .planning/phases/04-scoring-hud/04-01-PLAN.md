  2. Create src/systems/score.js with module-level closure pattern (consistent with lives.js):
     - `let score = 0;`
     - `let currentLevel = 1;` — placeholder until Phase 5 provides real level data
     - `createScore()` — subscribes to events:
       - OBSTACLE_DESTROYED: `score += SCORE.DESTROY_POINTS * currentLevel`

     - `resetScore()` — sets score to 0 and currentLevel to 1
     - `updateScore(dt)` — while game is playing, adds `SCORE.SURVIVAL_RATE * dt` to score (accumulates fractional, getScore floors it)
     - `getScore()` — returns `Math.floor(score)` (integer display value)

  Use the same import pattern as lives.js: import events from '../core/events.js' and SCORE from '../config.js'.
  Do NOT import state.js (survival update gating is handled by the caller in main.js — updateScore is only called when PLAYING).

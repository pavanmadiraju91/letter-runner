<task type="auto">
  <name>Task 1: Level-up announcement system</name>
  <files>src/systems/level-announce.js</files>
  <action>
Create src/systems/level-announce.js:

Module-level state (consistent with other systems' closure pattern):
- let active = false
- let timer = 0
- let level = 0
- const DURATION = 1.5 (seconds)

createLevelAnnounce():
- Subscribe to LEVEL_UP event: set active=true, timer=DURATION, level=payload.level
- Subscribe to GAME_RESTART event: set active=false, timer=0 (cancel any in-progress announcement)

updateLevelAnnounce(dt):
- If not active, return
- timer -= dt
- If timer <= 0: active = false

renderLevelAnnounce(ctx, width, height):
- If not active, return
- Calculate alpha: timer / DURATION (1.0 at start, fades to 0)
- Apply slight ease: alpha = alpha * alpha (quadratic ease-out feels snappier)
- ctx.save()
- ctx.globalAlpha = alpha
- ctx.fillStyle = '#ffffff'
- ctx.font = 'bold 64px monospace'
- ctx.textAlign = 'center'
- ctx.textBaseline = 'middle'
- ctx.fillText('LEVEL ' + level, width/2, height/2)
- Optionally draw a subtle glow: ctx.shadowColor = '#00ffcc', ctx.shadowBlur = 20 (before fillText)
- ctx.restore()
  </action>

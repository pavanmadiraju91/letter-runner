<task type="auto">
  <name>Task 2: Create index.html and config module</name>
  <files>index.html, src/config.js</files>
  <action>
1. Create `index.html` in project root:
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Letter Runner</title>
  <style>
    * { margin: 0; padding: 0; box-sizing: border-box; }
    html, body { width: 100%; height: 100%; overflow: hidden; background: #0a0a0f; }
    canvas {
      display: block;
      image-rendering: pixelated;
      image-rendering: -moz-crisp-edges;
    }
  </style>
</head>
<body>
  <canvas id="game"></canvas>
  <script type="module" src="/src/main.js"></script>
</body>
</html>
```

Key CSS decisions:
- `* { margin: 0; padding: 0 }` removes browser defaults (no scrollbars)
- `overflow: hidden` prevents any scroll
- `background: #0a0a0f` matches canvas fill for no flash of white
- `image-rendering: pixelated` for crisp pixel art on the canvas element
- `display: block` removes inline gap below canvas

2. Create `src/config.js`:
```js
export const COLORS = {
  BG: '#0a0a0f',
  DEBUG_TEXT: '#3a3a4a'
};

export const GAME = {
  TARGET_FPS: 60,
  MAX_DT: 1 / 30  // Cap delta-time to prevent spiral-of-death
};
```

3. Create a placeholder `src/main.js` so Vite can resolve the script tag:
```js
// src/main.js — placeholder until Plan 02 implements the full bootstrap
const canvas = document.getElementById('game');
const ctx = canvas.getContext('2d');

// Temporary: fill viewport with dark background to prove canvas works
function resize() {
  const dpr = window.devicePixelRatio || 1;
  canvas.width = window.innerWidth * dpr;
  canvas.height = window.innerHeight * dpr;
  canvas.style.width = window.innerWidth + 'px';
  canvas.style.height = window.innerHeight + 'px';
  ctx.scale(dpr, dpr);
  ctx.fillStyle = '#0a0a0f';
  ctx.fillRect(0, 0, window.innerWidth, window.innerHeight);
}

resize();
window.addEventListener('resize', resize);
```

This placeholder will be replaced by Plan 02 with the proper module imports.
